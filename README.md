# Smart Emergency Response and Accident Assistance System

## Project Overview
A web-based emergency response system developed as a BCA final-year project. It helps users send emergency requests and allows Ambulance, Police, Hospitals, and Admin to manage and respond efficiently.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Modules
- User
- Admin
- Police
- Ambulance
- Hospital

## Features
- User Registration & Login
- Emergency Request System
- Role-Based Dashboard
- Request Tracking
- Hospital & Ambulance Management
- Admin Panel
